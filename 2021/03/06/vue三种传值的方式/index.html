6.vue三种传值的方式

vue中最传统的三种传值方式

1.父向子传值

第一步

在子组件的调用标签上, 添加一个自定义属性, 然后通过v-bind指定一个父组件的data数据作为自定义属性的值

第二步

在子组件的对象中, 添加一个props属性, 来接收父组件传递过来的数据

可对传过的值进行一个 验证，验证数据的合法性

  • type属性: 进行类型校验 
     可选值: Number , String , Boolean , Array , Object
  • required: 属性是否必须 
     可选值: true , false
